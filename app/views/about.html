Why use this app?

The Suunto Ambit range are great multisport watches, however the suport for structured workouts is limitied compared to competing watches. There is a built-in interval timer, but it only allows you to specify high and low duration, either based on distance or time. You cannot specify a target (pace, heart rate, cadence), and each interval needs to be of the same duration.

The built-in interval timer will not help you do a structured run session such as:

  • 15min Warm Up
  • 1200m @ 04:00 pace (400m RI)
  • 2x800m @ 03:50 pace (200m RI)
  • 4x400m @ 03:30 pace (200m RI)
  • 15min Cool Down
Adding support for structured workouts like this is the goal of the Interval App Generator.

Suunto Ambit Apps

One of the cool features of the Ambit watches is the ability to create custom apps for the watch. The apps can do anything from calculating how many beers you have burned off during your exercise, to guiding you through some high intensity intervals.

So even if the built-in interval timer has limitied functionality, structured workouts can be implemented as custom apps. There is an existing Interval App Generator that let you specify a set of intervals with different durations (based on time or distance), but it does not let you specify a target for each interval (pace, heart rate, cadence). This is something this generator adds.
